## `mnft.ownerOf(tokenId)`

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| **tokenId** | `bigint` | Yes | Machine NFT token ID |

### Returns

`string`: checksummed owner address.

### TypeScript

```typescript
const owner = await sdk.mnft.ownerOf(BigInt(sdk.getManifest().machineTokenId));
```
