## `onchainid.isVerified(IsVerified)`

Read-only check against the vault's Identity Registry (`isVerified`).

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| **wallet** | `string` | Yes | EOA to check |

### Returns

| Field | Type | Description |
|-------|------|-------------|
| **verified** | `boolean` | KYC/compliance status |
| **wallet** | `string` | Checksummed address |

### TypeScript

```typescript
const { verified, wallet } = await sdk.onchainid.isVerified({
  wallet: sdk.getManifest().alice,
});
```
